Sobre o projeto

Este projeto foi desenvolvido utilizando Dream Maker (BYOND) com o objetivo de criar um sistema funcional de personagem para jogos online.
O foco foi construir uma base sólida que permite interação, personalização e estrutura de jogador dentro de um mundo persistente.

Funcionalidades
Sistema de login de jogador
Criação de personagem
Customização visual:
Cabelo
Estrutura de dados do personagem
Atualização dinâmica da aparência (overlays)

Tecnologias utilizadas
Dream Maker (BYOND)
Lógica de programação orientada a objetos
Manipulação de overlays e atributos
Exemplo:
Tela de login
Menu de customização
Personagem no jogo

 O que este projeto demonstra

Este projeto evidencia habilidades em:
Estruturação de sistemas de jogo
